# Hospital Healthcare Analytics
### Project Overview
This project showcases an interactive Healthcare Analytics Dashboard built in Power BI to analyze patient flow, hospital operations, bed utilization, and patient experience. The dashboard is designed to support executive-level decision-making through clear KPIs and actionable insights.

### Key Features
•	Hospital summary KPIs (patients, admissions, discharges, LOS, ER time)
•	Patient demographics by age, gender, and location
•	Department-wise ER time and inpatient analysis
•	Bed occupancy and utilization tracking
•	Revenue and patient satisfaction insights
•	Time-based analysis using a calendar dimension

### Data & Modeling
•	Cleaned and transformed raw Excel data using Power Query
•	Implemented a star schema with a central fact table and supporting dimensions
•	Built optimized one-to-many relationships with single-direction filtering
•	Created DAX measures for averages, counts, utilization, and performance metrics

### Tools Used
•	Power BI
•	Power Query
•	DAX
•	Microsoft Excel

### Outcome
The dashboard provides a consolidated view of healthcare operations, helping identify bottlenecks, optimize resources, and improve patient care through data-driven insights.
